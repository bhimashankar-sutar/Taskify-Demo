configure Git 
git config --global user.name "Name"
git config --global user.email "email"
git config --list - > to get list of all configuration

cd : change directory
ls : list files in directory
ls -a : for hidden files
clear : clear command

basic git command
1) clone
 git clone <repo code link>

2) status
git status - showing status with branch

types of status
untracked - new file not in git created in editor
modified - changed in file
unmodified  - unchanged
staged - file ready to be committed means it is added but not committed.

Note - when file is modified we need to add and commit the changes 

add .-
git add <file name> (. means all file)

commit
git commit -m "some message"

Push
git push origin main

Git branches:

1. git branch: to check branch 

2. git checkout<branch name>: to navigate another branch 


3. git checkout -b < new branch name>--> to create new branch

4. git branch -d <branch name>: to delete branch
